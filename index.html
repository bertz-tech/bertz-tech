  • Linux OS Linux è kernel che viene distribuito in una moltitudine di sistemi operativi. Famoso per la sua stabilità e sicurezza è apprezzato da molti per le configurazioni lato server e la sua tipica inclinazione al rimanere un prodotto per esperti del software.

  • Windows OS Windows è il più noto sistema operativo del pianeta. Sebbene sia usato da molti per scopi d'ufficio rimane un caposaldo nel settore dello sviluppo informatico e del tech. E potenzialmente corretto dire che, essendo "non free" e non "open source", la maggior parte degli sviluppi avvengo grazie a tecnologie trasversali e cross compatibili come le web technologies.e

  • Apple OS IOS (iPhone, iPod, iPad e iMac) è il sistema operativo distribuito non free per l'uso e sviluppo sulle piattaforme Apple. E potenzialmente corretto dire che, essendo "non free" anche nell'hardware su cui si appoggia, la maggior parte degli sviluppi avvengo grazie a tecnologie trasversali e cross compatibili come le web technologies, testate poi su macchine virtuali ove verrà installato tale sistema operativo.

  • Adroid OS Android & ChromeOS (SmartPhone e ChromeBook) è tra i sistemi operativi ultimi sul mercato. E potenzialmente corretto dire che, pur esendo in AOSP license, quindi open sources, essendo i suoi sviluppi limitati agli smartphone android, la maggior parte degli sviluppi avvengo grazie a tecnologie trasversali e cross compatibili come le web technologies.

  • Cli / Bash / Terminal ... Fondamentalmente è il modo in cui un programmatore accede, per vie minimaliste e direttissime, ai comandi di un sistema operativo o programma informatico scrivendo direttamente su un "terminale", un pannello privo di fronzoli estetici che bypassa le UI (per brevità: le grafiche) e diversi controlli.

  • HTML L'HTML (HYPERTEXT MARKUP LANGUAGE); Al contrario di quanti molti pensano, l'html non è un linguaggio di programmazione ma un sistema di markup atto a definire e raggruppare elementi seguiendo uno schema/semantica precisa e prestrutturata.

  • CSS Il CSS (Cascade stylesheet) è un protolinguaggio che serve a definire i paramentri visivi di una struttura ad oggetti o di markup come l'HTML.

  • Javascript Javascript è l'attuale linguaggio di programmazione web per antonomasia. Nato per gestire le interazioni degli utenti nelle pagine web è ora un complesso linguaggio di prototipazione e per server con cui si agisce nella quasi totalità delle app e siti di alto livello.

  • PHP PHP è un linguaggio di programmazione storico del web. Ad oggi alla versione 8 è anche il principale linguaggio con cui è strutturato wordpress e la maggior parte dei server low cost del pianeta.

  • API Le API (Application Program Interface) sono un elaborato - spesso in javascript o php - per la comunicazione dati tra un client (ad esempio il tuo computer) e un server online (un computer connesso al web che stai interrogando su cosa ti serve). Essendo spesso un set di funzioni direttamente connesse a database ed interazioni, le API sono una parte integrante della logica ed il funzionamento di un sofware moderno.

  • Node Js Node Js è un "sistema" (runtime system) open source multipiattaforma che permette l'esecuzione di javascript in tempo reale e lato server. E' la moderla ed eleastica soluzione dello sviluppo backend e software in opposizione allo storico e marmoreo modello di php.

  • Npm Npm è il gestore dei pacchetti, progetti e plugin per node js.

  • Mongo DB In opposizione ai database a tabelle dell'ecosistema php, mongoDB è il database scalabile di nuova generazione capace di garantire performance di sviluppo e richiesta senza pari grazie al modello a documenti/oggetto simile a javascript.

  • Cordova / Phonegap Cordova è l'antenato di "Capacitor" ed è stato uno tra i primi compilatori in grado di portare i linguaggi del web nello sviluppo app di android e ios. Purtroppo il progetto, preso in mano da Adobe e rinominato Phonegap... sembra essere definitivamente fallito.

  • PWA Le pwa (Progressi Web App) sono, in sintesi, i sofware di nuova generazione realizzati con il linguaggi del web e conformi ad una specifica filosofia google

  • Wordpress Wordpress è la piattaforma in php più espansa e venduta del pianeta. Rappresenta il 37% dei siti nel mondo e tra i sistemi low cost più venduti di tutti i tempi.

  • Wamp Wamp è un ecosistema server, in windows, per la gestione dei siti internet in locale (sul proprio computer).

  • VSCode Visual studio code è l'editor dei programmatori web -e non solo- del momento. Ed è l'editor con il quale è stato scritto questo sito.

  • Atom Editor Atom è il vecchio editor di codice fornito da Github. Nonostante la meravigliosa integrazione con la piattaforma, allo stato attuale, sembra esserci una vera e propria continua migrazione verso Visual studio code.

  • CSharp C# o CSharp è un linguaggio di programmazione tipicizzato ed altamente strutturato di microsoft. E' anche un linuguaggio per lo sviluppo web ma, in molti studi come il nostro, è utilizzato in Unity Engine per la realizzazione di componenti per il settore videoludico.

  • Unity Engine Unity 3D è un engine per il settore di sviluppo videludico. Un ecosistema per la realizzazione del multimedia entertainment.

  • SEO e Semantic Web Quando si parla di indicizzare un sito o una pagina web, si parla di una serie di ottimizzazioni per i motori di ricerca (SEO Search-Engine-Optimization). Il miglior modo per fare SEO rimane quello di eseguire un ottimo lavoro di costruzione del sito web, per fare ciò occorre un'eccellente "semantica" nel codice e nel contenuto. In estrema sintesi si potrebbe definire la semantica di un sito come l'insieme di un ottimo codice, un ottima struttura html che definisca al meglio un ottimo contenuto (Copy Seo Oriented) con in più le ottimizzazioni tecniche tipiche come "META", "AMP", "W3C", "SCHEMA.ORG" [...] altro.

  • SEM & ADSENSE SEM, Search engine marketing... letteralmente il marketing applicato al web. Tra i doveri più comuni ci sono la pianificazione ed il controllo di campagne di view e ppc su google adsense online.

  • FIGMA DESIGN Figma Design è il software cross piattaforma per la prototipazione, in grafica vettorale, più vicina alla vera realizzazione di un sito web o app.

  • Adobe suite Sopratutto Illustrator (grafica vettoriale), Photoshop (Fotoritocco e CGI), Premiere (Video Editing)... una delle più famose triadi della grafica dal 1990 ad oggi per lo sviluppo di un brand. Ovviamente consigliamo di visitare il sito ufficiale per comprendere l'enormità della suite Adobe.

  • Blender Open source e free ma forse, con Cinema 4D, il miglior software per la realizzazione di grafiche 3d a 360° sul pianeta.

  • e di più... Semplicemente, non si finisce mai di imparare e mai di completare questa lista.
